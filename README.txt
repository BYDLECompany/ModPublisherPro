# Mod Publisher Pro

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)

[![twitter](https://img.shields.io/badge/discord-1DA1F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.gg/B8DTMvmxye)

Mod Publisher Pro is an open-source tool designed to streamline the process of publishing mods to the BYDLE Client for DunHero. This software provides an intuitive interface and powerful features that make mod management and distribution easier for developers and players alike.

## Features
- Easy Mod Upload: Upload your mods to the BYDLE Client with a simple and guided process.
- Version Control: Keep track of mod versions and updates with built-in version management tools.
- Compatibility Checking: Automatically check for compatibility with the current version of DunHero.
- User-Friendly Interface: An intuitive interface that simplifies the mod publishing process for developers of all levels.
- Customizable Mod Metadata: Add and edit metadata such as mod description, author information, and more.
- Error Handling: Robust error handling and validation checks ensure smooth mod publishing.
- Batch Uploads: Support for batch uploading multiple mods at once.

## Installation
To install Mod Publisher Pro, follow these steps:

1. Clone the repository
```bash
git clone https://github.com/BYDLECompany/ModPublisherPro.git
cd ModPublisherPro
```

2. Install the required dependencies
```bash
pip install -d requirements.txt
```

3. Run the application
```bash
python main.py
```

Alternatively, you can download the latest release from the [Releases](https://github.com/BYDLECompany/ModPublisherPro/releases) page.

## Usage
1. Open Mod Publisher Pro and sign in with your BYDLE Client account.
2. Select the mod you want to publish from your local files.
3. Fill in the required metadata fields (e.g., mod name, version, description).
4. Check for any compatibility issues.
5. Click "Publish" to upload the mod to the BYDLE Client.
